# 📘 AZ-104: Introduction and Course Overview

## 🎯 Objective
Provide an overview of the AZ-104 certification, its scope, the core skills required for Azure administration, and the structure of the learning path.

---

## ☁️ Overview of Azure Administration

Microsoft Azure is a cloud computing platform used to deploy, manage, and monitor infrastructure and services in a cloud environment. It supports administration of resources such as:
- Virtual machines
- Storage solutions
- Networking components
- Identity and access services

The AZ-104 certification is designed for Azure Administrators responsible for implementing, managing, and monitoring an organization’s Azure environment.

---

## 🚀 Importance of AZ-104

AZ-104 is important because:
- Cloud technologies are widely adopted across industries
- Organizations need professionals who can manage cloud infrastructure effectively
- Azure administration skills are relevant for modern IT operations
- Cloud platforms are increasingly replacing or extending traditional on-premises environments

This certification is especially useful for those moving from roles involving:
- Server administration
- Network management
- Desktop and endpoint support
- On-premises infrastructure operations

---

## 🧠 Learning Approach

Successful preparation for AZ-104 requires:
- Understanding core Azure concepts
- Working with Azure services through practical exercises
- Applying concepts to real-world scenarios
- Building hands-on experience in Azure Portal

The certification is strongly practical in nature, and many questions are based on implementation and administrative scenarios rather than theory alone.

---

## 📘 Course Scope

The AZ-104 learning path is structured around the official certification blueprint and focuses on the major responsibilities of an Azure Administrator.

The course is designed to cover:
- Identity and governance
- Virtual networking
- Storage
- Compute resources
- Monitoring and backup

It also emphasizes practical administrative tasks such as:
- Managing storage resources
- Configuring virtual networks
- Understanding VPN connectivity
- Deploying and managing resources through Azure Portal

---

## 🧩 Core Functional Areas

### 1. Identity and Governance
- Managing users and groups
- Role-Based Access Control (RBAC)
- Governance and administrative control
- Azure Active Directory / Microsoft Entra ID

### 2. Networking
- Virtual Networks (VNet)
- Subnets
- IP addressing
- VPN and connectivity services

### 3. Storage
- Blob storage
- File storage
- Managed disks
- Storage management scenarios

### 4. Compute
- Virtual Machines (VMs)
- Compute resource deployment
- Resource management and availability

### 5. Monitoring and Backup
- Monitoring Azure resources
- Logs and alerts
- Backup and recovery operations

---

## 🛠️ Resource Management Methods

Azure resources can be managed using multiple approaches:
- Azure Portal
- Azure CLI
- PowerShell
- ARM Templates
- Microsoft Entra administration tools

These methods support different administration styles, including graphical, command-line, and infrastructure-as-code approaches.

---

## 📚 Prerequisites

Before starting AZ-104, it is helpful to have a basic understanding of:
- Cloud computing concepts
- Networking fundamentals
- Server and computer administration
- Virtualization concepts
- Operating systems such as Windows or Linux

This foundational knowledge makes it easier to understand how Azure services map to traditional infrastructure concepts.

---

## 🌍 Foundational Concepts

Before moving deeper into Azure administration, it is important to understand:
- What cloud computing is
- Cloud vs On-Premises vs Virtualization
- Azure Subscriptions
- Azure Regions and Availability Zones
- Azure free trial setup
- Azure Portal interface and navigation

These topics form the starting point for understanding how Azure services are organized and managed.

---

## 🧪 Exam Preparation Focus

Preparation for AZ-104 should include:
- Reviewing the official certification skills outline
- Practicing in Azure Portal
- Understanding scenario-based questions
- Building familiarity with real administrative tasks

A strong hands-on approach is important because Azure certifications commonly test practical decision-making and implementation knowledge.

---

## 🔄 Why Updated Learning Matters

Azure is an evolving cloud platform, and services, interfaces, and administrative workflows can change over time. Current learning should align with the latest Azure portal experience and updated certification scope.

For this reason, it is important to study using an up-to-date topic structure and modern Azure service terminology.
