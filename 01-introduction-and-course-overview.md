# 📘 AZ-104: Introduction and Course Overview

## 🎯 Objective
Provide an overview of the AZ-104 certification, its scope, required skills, and the structure of topics covered.

---

## ☁️ Overview of Azure Administration

Microsoft Azure is a cloud computing platform used to manage and deploy infrastructure such as:
- Virtual machines
- Storage solutions
- Networking components
- Identity and access management systems

The AZ-104 certification focuses on the skills required to manage these resources in a cloud environment.

---

## 🚀 Importance of AZ-104

- Increasing adoption of cloud technologies across industries
- Demand for professionals skilled in managing cloud infrastructure
- Transition from on-premises environments to cloud-based solutions
- Requirement for managing services through centralized platforms like Azure Portal

---

## 🧠 Learning Approach

Effective preparation requires:
- Hands-on practice using Azure Portal
- Understanding real-world scenarios
- Applying concepts in practical environments

Most certification questions are based on implementation scenarios rather than theoretical knowledge.

---

## 🧩 Core Functional Areas

### 1. Identity and Governance
- Managing users and groups
- Role-Based Access Control (RBAC)
- Azure Active Directory (Microsoft Entra ID)

### 2. Networking
- Virtual Networks (VNet)
- Subnets and IP addressing
- Connectivity solutions such as VPN

### 3. Storage
- Blob storage
- File storage
- Managed disks

### 4. Compute
- Virtual Machines (VMs)
- Resource deployment and management
- Scaling and availability

### 5. Monitoring and Backup
- Monitoring resource performance
- Logging and alerts
- Backup and disaster recovery

---

## 🛠️ Resource Management Methods

Azure resources can be managed using multiple approaches:
- Azure Portal (Graphical Interface)
- Azure CLI
- PowerShell
- ARM Templates (Infrastructure as Code)

---

## 📚 Prerequisites

Basic understanding of the following is recommended:
- Cloud computing concepts
- Networking fundamentals (IP addressing, routing)
- Operating systems (Windows/Linux)
- Virtualization concepts

---

## 🌍 Foundational Concepts

Before progressing further, it is important to understand:
- Cloud vs On-Premises vs Virtualization
- Azure Subscriptions
- Regions and Availability Zones
- Azure Portal interface

---

## 💡 Key Takeaways

- Azure is a leading cloud platform for managing IT infrastructure
- AZ-104 focuses on practical administration skills
- Real-world scenario understanding is critical for success
- Hands-on experience is essential for both learning and certification

---

## 🔁 Summary

- AZ-104 is designed for Azure Administrators
- Covers identity, networking, storage, compute, and monitoring
- Emphasizes practical implementation over theory
- Requires consistent hands-on practice
